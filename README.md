# 📱 Tela de Login e Cadastro

Projeto desenvolvido em **React Native + Expo + TypeScript**, contendo uma interface com telas de Login e Cadastro.

## Funcionalidades

 Tela de Login
 Tela de Cadastro
 Componentes reutilizáveis
 Interface responsiva
 Navegação entre telas
 Desenvolvido com Expo Router

## Tecnologias

- React Native
- Expo
- TypeScript
- Expo Router

##  Estrutura do Projeto

```
src/
 ├── app/
 │   ├── index.tsx
 │   ├── signup.tsx
 │   └── layout.tsx
 ├── components/
 │   ├── Button.tsx
 │   └── Input.tsx
 └── assets/
```

##  Como executar

Clone o projeto:

```bash
git clone https://github.com/honocode/Tela-de-cadastro-e-login.git
```

Entre na pasta:

```bash
cd Tela-de-cadastro-e-login
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npx expo start
```
